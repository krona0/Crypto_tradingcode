# Crypto_tradingcode
几个用于加密货币交易策略，模型数学计算的代码仓库。

## btc_invsell.py

invsell 模型解决在市场高点减持过少或市场低点减持过多的问题，适用于每次BTC减半行情之后的单边行情，或者是波段交易。模型基于数学公式的分步减仓策略，结合了固定减仓比例和价格增幅目标，逐步减小持仓，同时记录减仓后的收益和剩余价值。

使用方法：
    在代码中更改参数运行
